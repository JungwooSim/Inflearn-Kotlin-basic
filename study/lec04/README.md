### 4강. 코틀린에서 연산자를 다루는 방법

- 단항 연산자 / 산술 연산자
- 비교 연산자와 동등성, 동일성
    - 코틀린에서는 Java 와 다르게 객체를 비교할 때 compareTo 를 호출해준다.
    - 동일성 & 동등성
        - Java : 동일성에 “==” 를 사용, 동등성에 equals 를 호출
        - Kotlin : 동일성에 “===” 를 사용, 동등성에 “==” 를 호출 ( “==” 를 사용하면 간접적으로 equlas 를 호출해준다)
- 논리 연산자 / 코틀린에 있는 특이한 연산자
- 연산자 오버로딩
